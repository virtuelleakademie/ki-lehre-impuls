# KI in der Lehre: Impuls

[![DOI](https://img.shields.io/badge/DOI-10.60877%2Fz2pq--qh10-blue)](https://doi.org/10.60877/z2pq-qh10)
[![Lizenz: CC BY 4.0](https://img.shields.io/badge/Lizenz-CC%20BY%204.0-lightgrey)](https://creativecommons.org/licenses/by/4.0/deed.de)

Eine Quarto-Website mit einem kompakten Impuls (Input, 30 Min.) zum aktuellen Stand von KI in der Hochschullehre. Die Inhalte sind auf Deutsch und richten sich an Mitarbeitende von Schweizer Hochschulen.

Website: <https://virtuelleakademie.github.io/ki-lehre-impuls/>
